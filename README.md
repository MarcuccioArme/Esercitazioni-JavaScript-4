# Esercitazioni-JavaScript-4
### Alcuni semplici esercizi per apprendere JavaScript

## Esercizio 1:
### "Validare i campi di un modulo di contatto"

Una pagina Web consente di accettare i dati relativi ad un “modulo di contatto clienti”, simile a quello qui allegato.
<br><br><img src="readmeSrc\1.png" alt="1" width="50%" height="50%" style="margin: 1;">

Realizzare il codice JavaScript che consenta:
- l’inserimento dei dati necessari;
- la spedizione del modulo nel caso vengano rispettati i controlli di validità seguenti (il clic su “Invia” deve segnalare che il modulo “non può essere inviato”, se anche uno solo dei controlli di validità sui dati, da esito “negativo”):
  - Nome e cognome -> NON NULLO;
  - Indirizzo -> NON NULLO;
  - CAP -> NON NULLO - DEVE ESSERE COMPOSTO DA 5 CARATTERI;
  - Comune -> NON NULLO;
  - Provincia -> NON NULLO - 2 CARATTERI MAX;
  - Telefono -> NON NULLO – NEL FORMATO (PREFISSO) – NUMERO es.: (080) - 54312345
  - Email -> DEVE CONTENERE SIMBOLO “@”
  - Telefono -> NON NULLO;
  - Messaggio -> NON NULLO
